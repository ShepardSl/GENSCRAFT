# GENSCRAFT — ГЕНШИН В МАЙНКРАФТЕ СЕРВЕР

Промо-страница MMORPG в сеттинге Genshin Impact.  
**Прод:** [https://genscraft.net](https://genscraft.net)

## 🚀 Запуск (локально)

Проект — статический сайт, бэкенд не нужен. Любой из способов:

```bash
# Вариант 1: npx serve
npx -y serve .

# Вариант 2: VS Code Live Server
# Установить расширение "Live Server" → ПКМ на index.html → Open with Live Server

# Вариант 3: Python
python -m http.server 3000
```

Откройте `http://localhost:3000` (или порт из Live Server).

## 📁 Структура

```
├── index.html              Главная страница
├── robots.txt              Правила для поисковых роботов
├── sitemap.xml             Карта сайта
├── README.md               Документация
└── assets/
    ├── css/
    │   └── styles.css      Основные стили
    ├── js/
    │   └── main.js         Скрипты (модалки, анимации, пасхалка)
    └── img/                Изображения, видео, иконки
```

## 🎮 Фичи

- Hero-секция с персонажем и CTA-кнопками
- Секция «О разработке» — карточки особенностей с анимацией появления
- Секция «Showcase» — демонстрация мира с встроенным YouTube-видео в кастомном фрейме
- Модальные окна: выбор способа оплаты доната и получение проходки
- Система копирования IP адреса с визуальным фидбеком
- Scroll-reveal анимация карточек состава мира
- Магический шлейф из частиц за курсором (в стиле Genshin)
- Плавная навигация по якорным ссылкам
- `prefers-reduced-motion` поддержка
- SEO: OG/Twitter мета-теги, canonical, Schema.org JSON-LD

## 🛠 Технологии

- HTML5, CSS3 (Vanilla), Vanilla JS
- Google Fonts (Roboto Condensed, Comfortaa)
- SVG-фильтры и маски для графических эффектов
- Без фреймворков и сборщиков
- Семантическая, адаптивная разметка
